# Security

If you discover a potential security issue in this project, please **do not create a public GitHub issue**.

Instead, report it through one of the following channels:

- Email [AWS Security](mailto:aws-security@amazon.com), or
- Use the [AWS Vulnerability Reporting](https://aws.amazon.com/security/vulnerability-reporting/) page.

Please do **not** include sensitive data (account IDs, ARNs, credentials, or customer
information) in your report.

See [CONTRIBUTING](CONTRIBUTING.md#security-issue-notifications) for more information.
